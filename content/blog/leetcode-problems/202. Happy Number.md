note: can also use the Floyd's cycle detection algo.

```python
class Solution:
    def isHappy(self, n: int) -> bool:
        seen = set()
        while n != 1 and n not in seen:
            seen.add(n)
            n = sum((int(ch) ** 2) for ch in str(n))

        return n == 1
```
