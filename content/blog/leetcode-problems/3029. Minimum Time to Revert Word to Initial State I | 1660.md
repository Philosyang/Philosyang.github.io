Surprisingly easy one.

```python
class Solution:
    def minimumTimeToInitialState(self, word: str, k: int) -> int:
        # equivalent ask:
        # the minimum x that word.startsWith(word[x*k:])
        # "abcdefg", k = 3
        # "defgxxx" -> "defg**a"
        # "gxxxyyy" -> "g**abcd"
        # "xyyyzzz" -> "abcdefg"

        def startsWith(word: str, prefix: str) -> bool:
            # "abcdefg", "abc"
            return word[: len(prefix)] == prefix

        n = len(word)
        result = n // k + 1
        possible_minimum = result - 1

        for possible_minimum in range(result - 1, 0, -1):
            if startsWith(word, word[possible_minimum * k :]):
                result = possible_minimum

        return result
```

I have realized that python string class has `startswith(self)` built in:

```python
class Solution:
    def minimumTimeToInitialState(self, word: str, k: int) -> int:
        n = len(word)
        result = n // k + 1
        possible_minimum = result - 1

        for possible_minimum in range(result - 1, 0, -1):
            if word.startswith(word[possible_minimum * k :]):
                result = possible_minimum

        return result
```
