The more meaningful takeaway from this question for me is a refresher on common String class methods in Python.

```python
>>> 'a'.isalnum()  # !
True
>>> 'a'.isalpha()
True
>>> 'a'.isnumeric()
False
>>> 'a'.lower()
'a'
>>> '1'.lower()  # !
'1'
>>> '1'.upper()
'1'
```

Two pointers (O(1) space) example solution:
```python
class Solution:
    def isPalindrome(self, s: str) -> bool:
        l, r = 0, len(s) - 1

        while l < r:
            a, b = s[l], s[r]

            if not a.isalnum():
                l += 1
                continue
            elif not b.isalnum():
                r -= 1
                continue

            if a.lower() != b.lower():
                return False

            l += 1
            r -= 1

        return True
```
