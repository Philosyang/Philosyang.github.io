Don't get fooled by the easy.

I initially went straight with non-popping + nlargest, that's O(nlogn) which TLEs.

```python
class KthLargest:

    def __init__(self, k: int, nums: List[int]):
        self.k = k
        self.elements = []
        for num in nums:
            self.add(num)

    def add(self, val: int) -> int:
        heapq.heappush(self.elements, val)
        if len(self.elements) > self.k:
            heapq.heappop(self.elements)
        return self.elements[0]
```
