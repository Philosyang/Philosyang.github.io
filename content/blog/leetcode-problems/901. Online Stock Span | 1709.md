It's a monotonic stack problem!

```python
class StockSpanner:

    def __init__(self):
        self.history = deque([])

    def next(self, price: int) -> int:
        result = 1
        while self.history and self.history[-1][0] <= price:
            result += self.history.pop()[1]

        self.history.append((price, result))
        return result
```
