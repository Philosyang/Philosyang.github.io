For some reason O(n**3) is passing:

```python
class Solution:
    def countSubstrings(self, s: str) -> int:
        # trying to find an efficient way to determine palindromic
        # sliding window doesn't seem to work
        # 'aba' -> 'a', 'ab', 'b', 'ba', 'a'; misses 'aba'
        # will brute-force TLE?
        if not s:
            return 0

        n = len(s)
        l, r = 0, 1  # [l, r)
        result = 0

        for l in range(n):
            for r in range(l + 1, n + 1):
                test = s[l:r]
                if test == test[::-1]:
                    result += 1

        return result
```

The missed idea here is that since we are looking for palindromes, we should iterate over all possible centers (O(n); actually 2n-1 because of odd & even length), and expand on each (O(n)).

```python
class Solution:
    def countSubstrings(self, s: str) -> int:
        # iterate all centers, then expand
        n = len(s)
        result = 0

        # odd len palindromes
        c = 0
        for c in range(n):
            arm_len = 0
            while c - arm_len >= 0 and c + arm_len < n:
                if s[c - arm_len] == s[c + arm_len]:
                    result += 1
                    arm_len += 1
                else:
                    break

        # even len palindromes
        c_left = 0
        for c_left in range(n - 1):
            arm_len = 0
            while c_left - arm_len >= 0 and c_left + 1 + arm_len < n:
                if s[c_left - arm_len] == s[c_left + 1 + arm_len]:
                    result += 1
                    arm_len += 1
                else:
                    break

        return result
```

Don't reuse code:

```python
class Solution:
    def countSubstrings(self, s: str) -> int:
        # iterate all centers, then expand
        n = len(s)
        result = 0

        def expand(l: int, r: int):
            count = 0
            while l >= 0 and r < n and s[l] == s[r]:
                count += 1
                l -= 1
                r += 1
            return count

        for c in range(n):
            result += expand(c, c)
            result += expand(c, c + 1)
        return result
```
